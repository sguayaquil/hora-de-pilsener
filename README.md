# Hora de Pilsener 🍺

A fun, interactive GitHub Pages project with a button that displays a celebratory message and image.

## What it does

- Click the "¿Que hora es?" button
- See the message "¡Siempre es hora de una Pilsener!" appear
- View a festive Pilsener image
- Click again to toggle the message and image

## GitHub Pages Setup

To deploy this as a GitHub Page:

1. Push this repository to GitHub
2. Go to your repository settings
3. Scroll to "GitHub Pages" section
4. Select **Source**: "Deploy from a branch"
5. Select **Branch**: "main" (or your default branch) and folder "**/root**" (or create a `docs/` folder)
6. Your page will be available at: `https://yourusername.github.io/hora-de-pilsener/`
